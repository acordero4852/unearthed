# UnEarthed

A gift catalog web app built for CodePath WEB103. Browse a collection of gifts, view individual gift details, and explore the catalog through a React frontend backed by an Express + PostgreSQL API.

## Tech Stack

- **Frontend:** React, React Router, Vite
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (`pg`)

## Project Structure

```txt
unearthed/
├── client/   # React + Vite frontend
└── server/   # Express + PostgreSQL backend
```

## Getting Started

### Server

```bash
cd server
npm install
npm run reset   # seed the database
npm start
```

### Client

```bash
cd client
npm install
npm run dev
```
